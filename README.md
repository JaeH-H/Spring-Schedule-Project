## Schedule-Project
Spring을 사용해서 일정을 관리를 구현했습니다.

## 프로젝트 소개
  * 각 일정의 고유 식별자(ID)를 자동으로 생성하여 관리
  * 작성자, 비밀번호, 할 일, 작성일/수정일을 저장
  * 전체 일정 조회(수정일 기준 내림차순), 선택 일정 조회
  * 선택 일정 수정, 선택 일정 삭제
